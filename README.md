# Dockerized Task Manager (ToDo List)

A full-stack ToDo List application built with **FastAPI**, **Streamlit**, and **PostgreSQL**, fully **dockerized** and ready for **CI/CD with GitHub Actions**.  

This project demonstrates a modern full-stack setup with containerization, automated testing, and deployment pipelines â€” perfect for learning DevOps and showcasing in your portfolio.

---

## Features

- Add, update, and delete tasks
- Mark tasks as **complete** or **incomplete**
- Modern UI with **Streamlit** and **TailwindCSS**
- Backend powered by **FastAPI** and **PostgreSQL**
- Fully **dockerized** for easy setup and deployment
- **GitHub Actions CI/CD** pipeline for automated testing and builds

---

## Tech Stack

| Layer         | Technology                  |
|---------------|-----------------------------|
| Backend       | FastAPI, Python, SQLAlchemy |
| Frontend      | Streamlit, TailwindCSS      |
| Database      | PostgreSQL                  |
| Containerization | Docker, Docker Compose    |
| CI/CD         | GitHub Actions             |
